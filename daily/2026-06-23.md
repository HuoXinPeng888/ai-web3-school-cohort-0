## 学习笔记

### 关键概念

**AI × Web3 School** 是一个面向 builders 的开源学习计划，核心关注：
- 模型如何理解任务、组织上下文、调用工具
- Agent 如何从"回答问题"进入"执行流程"
- 钱包和账户如何表达权限、额度、时间和撤销条件
- 链上交易如何模拟、确认、记录和审计
- 支付、身份、声誉、治理如何接入自动化系统
- 隐私、安全和可验证执行

### Learning Agent 的作用

Learning Agent 的目标是帮助学员：
1. 理解课程
2. 规划每日任务
3. 维护个人学习仓库
4. 生成打卡草稿
5. 提醒同步到 WCB/打卡平台
6. 把学习问题沉淀为可开源、可索引、可复盘的材料

### MetaMask 钱包创建教程

#### 方式一：浏览器扩展（推荐用于开发）

**步骤 1：访问官方网站**
打开浏览器，访问 MetaMask 官方网站：https://metamask.io/

**步骤 2：安装扩展**
点击 "Download" 按钮，选择你的浏览器（Chrome、Firefox、Brave、Edge），跳转到浏览器应用商店安装扩展。

**步骤 3：创建钱包**
1. 安装完成后，点击浏览器工具栏中的 MetaMask 图标
2. 点击 **"开始使用"**
3. 选择 **"创建新钱包"**
4. 设置钱包密码（至少8位，记住这个密码！）
5. 阅读并同意条款

**步骤 4：保存助记词（最重要！）**
1. 系统会显示 12 个英文单词的助记词（Seed Phrase）
2. **务必将这些单词按顺序记录在安全的地方**
3. 可以写在纸上，或使用硬件钱包存储
4. **永远不要将助记词保存在联网设备上！**
5. 验证助记词：按顺序点击单词确认

**步骤 5：完成设置**
钱包创建完成，你会看到钱包地址和余额界面。

#### 方式二：移动端 App

1. 在 App Store 或 Google Play 搜索 "MetaMask" 下载安装
2. 打开应用，点击 **"创建钱包"**
3. 设置密码，保存助记词
4. 验证助记词完成创建

#### 安全提醒

| 注意事项 | 说明 |
|---------|------|
| **助记词安全** | 这是恢复钱包的唯一方式，丢失将无法找回资产 |
| **密码保护** | 密码用于解锁钱包，但不能恢复钱包 |
| **官方渠道** | 只从 metamask.io 或官方应用商店下载 |
| **谨防钓鱼** | 注意检查网址，不要泄露助记词给任何人 |
