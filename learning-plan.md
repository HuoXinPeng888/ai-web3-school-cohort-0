# 学习计划

## 学习路径

基于学员画像（AI 有基础 + Web3 新手 + 能独立开发），推荐以下学习路径：

### 阶段一：Web3 基础入门（第 1-2 周）

**目标**: 建立对区块链、钱包、智能合约的基本认知

- [ ] 理解区块链基本概念
- [ ] 创建并使用钱包（MetaMask 等）
- [ ] 了解以太坊/EVM 基础
- [ ] 熟悉 Remix IDE 开发环境
- [ ] 学习 Solidity 基础语法
- [ ] 完成第一个智能合约部署（Remix VM）
- [ ] 了解测试网和测试币（可选，后续再弄）
- [ ] 连接 MetaMask 到测试网并部署合约

### 阶段二：AI Agent 基础（第 3-4 周）

**目标**: 掌握 AI Agent 开发核心技能

- [ ] Prompt Engineering 进阶
- [ ] Agent 架构设计
- [ ] 工具调用（Tool Calling）
- [ ] RAG 基础
- [ ] 构建第一个 AI Agent

### 阶段三：AI × Web3 交叉（第 5-6 周）

**目标**: 将 AI 与 Web3 结合

- [ ] AI Agent + 钱包集成
- [ ] AI Agent + 智能合约交互
- [ ] 链上数据读取与分析
- [ ] 可验证执行

### 阶段四：Hackathon 项目（第 7-8 周）

**目标**: 完成一个完整的 AI × Web3 项目

- [ ] 确定项目方向
- [ ] 设计架构
- [ ] 开发 MVP
- [ ] 测试与优化
- [ ] 提交展示

## 每日学习流程

1. **早上**: 查看 WCB Learning 页面今日任务
2. **学习**: 阅读 Handbook 相关章节 + 实践
3. **记录**: 更新 `daily/YYYY-MM-DD.md`
4. **打卡**: 生成打卡草稿，提交到 WCB 平台
5. **反馈**: 记录问题到 `handbook-feedback/`
